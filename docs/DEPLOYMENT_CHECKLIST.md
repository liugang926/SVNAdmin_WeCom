# 企业微信集成部署检查清单

## 📋 部署前准备

### 环境检查

- [ ] **PHP 版本**: 7.2+ (推荐 8.0+)
- [ ] **PHP 扩展**: curl, json, pdo, sqlite3/mysqli, mbstring, openssl
- [ ] **数据库**: SQLite 3.0+ 或 MySQL 5.7+
- [ ] **Web 服务器**: Apache 2.4+ 或 Nginx 1.16+
- [ ] **SVN 版本**: Subversion 1.8+
- [ ] **操作系统**: Linux (CentOS 7+, Ubuntu 18.04+) 或 Windows Server

### 权限检查

- [ ] **文件权限**: SVNAdmin 目录可读写
- [ ] **日志权限**: `02.php/logs/` 目录可写
- [ ] **PID 权限**: `02.php/templete/pid/` 目录可写
- [ ] **配置权限**: `02.php/config/` 目录可写
- [ ] **SVN 权限**: SVN 仓库钩子目录可写

### 企业微信准备

- [ ] **企业微信账号**: 已认证的企业微信账号
- [ ] **管理员权限**: 具有应用管理权限
- [ ] **应用创建**: 已创建自建应用
- [ ] **通讯录权限**: 应用具有通讯录管理权限
- [ ] **Webhook 地址**: 已获取群机器人 Webhook 地址

## 🔧 安装步骤

### 1. 自动安装

- [ ] 运行安装脚本: `php 02.php/server/wecom_install.php install`
- [ ] 检查安装结果: 无错误信息
- [ ] 验证数据库表: 企业微信相关表已创建
- [ ] 确认配置文件: `02.php/config/wecom.php` 已生成

### 2. 配置向导

- [ ] 运行配置向导: `php 02.php/server/wecom_setup_wizard.php`
- [ ] 输入企业微信应用信息: CorpId, Secret, AgentId
- [ ] 配置同步选项: 同步间隔、匹配规则等
- [ ] 配置通知选项: 默认 Webhook、消息模板等
- [ ] 测试 API 连接: 连接测试成功

### 3. SVN 钩子安装

- [ ] 复制钩子脚本到 SVN 仓库
- [ ] 设置执行权限: `chmod +x hooks/post-commit`
- [ ] 测试钩子脚本: 手动执行无错误
- [ ] 验证钩子配置: 钩子名称和描述正确

## 🚀 服务启动

### 守护进程启动

- [ ] 启动主守护进程: `php 02.php/server/svnadmind.php start`
- [ ] 启动通知守护进程: `php 02.php/server/wecom_notification_daemon.php start`
- [ ] 检查进程状态: `ps aux | grep -E "(svnadmind|wecom_notification)"`
- [ ] 验证 PID 文件: PID 文件已创建且内容正确

### 系统服务配置 (可选)

- [ ] 创建 systemd 服务文件
- [ ] 启用自动启动: `systemctl enable svnadmin-wecom`
- [ ] 测试服务重启: `systemctl restart svnadmin-wecom`
- [ ] 验证服务状态: `systemctl status svnadmin-wecom`
