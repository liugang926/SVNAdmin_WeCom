## 🚀 核心功能

### 1. 组织架构同步

- **自动同步**: 企业微信部门结构 → SVN 用户组
- **层级保持**: 维护部门层级关系
- **智能映射**: 部门名称到用户组名的智能转换
- **增量更新**: 只同步变更的数据

### 2. 用户身份管理

- **多策略匹配**: userid、邮箱、手机号、姓名匹配
- **自动创建**: 可选的 SVN 用户自动创建
- **信息同步**: 用户基础信息同步到 SVN
- **状态管理**: 用户启用/禁用状态同步

### 3. 权限动态管理

- **基于部门**: 根据部门关系自动分配权限
- **层级继承**: 支持权限层级继承
- **灵活配置**: 支持仓库级和路径级权限配置
- **实时更新**: 权限变更实时生效

### 4. 实时消息通知

- **事件触发**: 支持 commit、delete、revprop-change 等事件
- **规则引擎**: 灵活的通知规则配置
- **消息模板**: 可自定义的消息格式
- **批量处理**: 高效的批量通知机制
- **失败重试**: 智能重试确保消息送达

### 5. Web 管理界面

- **现代化 UI**: 基于 Vue.js 的响应式界面
- **功能完整**: 5大管理模块
- **实时监控**: 同步状态和通知统计
- **用户友好**: 直观的操作界面
